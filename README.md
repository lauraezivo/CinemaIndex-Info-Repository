# CinemaIndex-Info-Repository
本仓库用于整理影视类网站的基础信息索引，主要记录电影、电视剧及相关影视站点的公开资料。所有信息均来自公开网络环境，仅用于信息归档与查询参考，不涉及任何内容分发或资源解析。仓库不提供播放、下载或跳转服务，也不参与站点运营。项目定位为中立的信息整理工具，方便后续对站点信息进行核查与更新。
